# Aegis Smart Stadium OS: Live Digital Twin Guide

This document details the live digital twin telemetry representation layer.

## Telemetry Mapping & Visualizations
- **Stadium Ring Bowl**: Graphical ellipse representing stands and concourses.
- **Crowd Density Heatmaps**: Responsive circle gradients reflecting stand occupancies.
- **Volunteer Steward Marks**: Indigo markers representing roster allocations.
- **Vehicle Routing Nodes**: Yellow rectangles tracking shuttle fleet movements.
- **CCTV simulated Stream**: Simulates feed telemetry from entrance cameras.
