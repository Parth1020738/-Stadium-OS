# Aegis Mission Control Dashboard

The Mission Control Dashboard page provides real-time visibility into the Multi-Agent AI coordination and digital twin telemetry.

## Navigation Path
The dashboard is accessible via `/mission-control` or the **Mission Control** link in the side navigation bar.

## Key Modules
1. **Active Agents Widget**: Monitors the real-time reasoning summaries, decision confidence, and focus actions of the 10 domain agents.
2. **Coordinator Synthesis**: Displays average coordination confidence, inference latency, and resolved agent conflicts.
3. **Digital Twin Sync**: Provides telemetry synchronizations for crowd capacity, active volunteers, incident logs, shuttle availability, and weather forecasts.
4. **Coordinated Timeline**: Chronologically structures planned operations steps (e.g. 00:00 - Open Gate D, 00:02 - Dispatch Volunteer Team Bravo).
5. **Command Approval Queue**: A secure interface supporting RBAC validation before any timeline command is executed.
6. **Executive briefings**: Compiles briefing cards for various stadium officials (CEO, Director of Operations, Transit Lead, Volunteer Coordinator, Accessibility Specialist).
