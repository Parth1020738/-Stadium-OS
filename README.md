# Aegis Smart Stadium OS

Aegis Smart Stadium OS is a hybrid edge-to-cloud AI platform designed to automate crowd safety, security dispatch, and personalized fan logistics for the FIFA World Cup 2026.

## Monorepo Layout

- `backend/` - FastAPI microservices (Python)
- `api-gateway/` - NestJS API Gateway (TypeScript)
- `frontend/` - React SPA (Vite/TS/Tailwind)
- `mobile/` - React Native App (Navigation + SQLite)
- `ai/` - LangGraph Agent framework
- `shared/` - Common modules for Node.js and Python
- `infrastructure/` - Dockerfiles, Compose setups, monitoring configs
- `scripts/` - Orchestration and setup scripts
- `docs/` - System architecture blueprints
- `tests/` - Testing foundations
- `.github/` - CI/CD GitHub Actions workflows
