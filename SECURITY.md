# Security Policy

Aegis Smart Stadium OS takes security seriously. This document details how to report vulnerabilities and our support policy.

---

## 1. Supported Versions

| Version | Supported |
| :--- | :--- |
| **v1.0.x** | :white_check_mark: Yes |
| **< v1.0.0** | :x: No |

---

## 2. Reporting a Vulnerability

**Please do not open public issues for security vulnerabilities.**

If you discover a potential security vulnerability, please report it privately by emailing the maintainer team at `security@aegis-os.org`.

Please include the following information in your report:
- A description of the vulnerability and its potential impact.
- Steps to reproduce the issue (including proof of concept scripts or screenshots where applicable).
- Any details regarding your environment or configuration.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible.
