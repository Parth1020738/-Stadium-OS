# AI Operations Prompt Library

Use these prompts inside the AI Copilot to check stadium telemetry and retrieve actionable recommendations.

## 1. Crowd Density & Gate Operations
- *"Why is Gate D crowded?"* (Inspects turnstile counts, CCTV density, scanner delays).
- *"Predict next congestion."* (Checks booking trends, arrivals telemetry, and door status).

## 2. Emergency & Incident Log Summary
- *"Summarize active incidents."* (Inspects open leak repairs, medical calls, scanner errors).
- *"What happened today?"* (Aggregates historical logs for the shift).

## 3. Staffing & Rosters
- *"Show volunteer shortages."* (Identifies under-allocated zones).
- *"Which volunteers are overloaded?"* (Flags active volunteers working double-shifts).

## 4. Transit & Shuttle Flows
- *"Predict transport delays."* (Checks maps traffic, vehicle telemetry).
- *"Recommend transport improvements."* (Suggests corridor bypass overrides).
